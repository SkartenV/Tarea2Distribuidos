Consideraciones:

- Actividad 1:

Esta actividad funciona completamente, sin embargo, no se ejecuta mediante Docker. Por lo que, en primer lugar se
debe abrir una consola dentro de la carpeta "Actividad1" y ejecutar el comando "python3 Servidor.py". Luego, en otra
consola dentro de la misma carpeta ejecutar el comando "python3 Cliente.py". En este momento, se debe introducir el
nombre del cliente y luego se desplegará un menú, pudiendo interactuar haciendo las acciones que son deseadas. Si se 
quieren abrir más clientes, se deben abrir otras consolas y ejecutar el comando "python3 Cliente.py", para así poder
enviar mensajes entre clientes. Al enviar un mensaje se pregunta a quién se le quiere enviar el mensaje para que
llegue correctamente al receptor.

Cuando a un cliente le llega un mensaje, lo recibe inmediatamente, mostrándolo en pantalla. Por lo que puede que
reciba un mensaje y se despliegue abajo del menú, dando a pensar que el programa se queda pegado, sin embargo, se
puede seleccionar otra opción del menú.

El archivo "log.txt" se encuentra en la carpeta "Actividad1".
